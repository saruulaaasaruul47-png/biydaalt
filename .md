# Cinema Ticket Booking System

## 1. Төслийн Танилцуулга

### Архитектур

REST API + MVC + Монолит

### Хэмжээ

Дунд хэмжээний систем (8 Entity, 30+ Endpoint)

### Зорилго

Кино театрын кино, танхим, үзвэрийн цаг, тасалбар захиалга болон хэрэглэгчийн удирдлагыг хийх REST API үйлчилгээ боловсруулах.

# 3. Функциональ Бус Шаардлага

## 3.1 Аюулгүй Байдал

* bcrypt password hashing
* JWT Access Token
* JWT Refresh Token
* RBAC
* Validation
* SQL Injection хамгаалалт
* CORS

---

## 3.2 Гүйцэтгэл

* API Response < 200ms
* Pagination бүх list endpoint дээр
* Database Index ашиглах
* JOIN Query оновчлох

---

## 3.3 Алдаа Боловсруулалт

HTTP Status Codes:

* 200 OK
* 201 Created
* 400 Bad Request
* 401 Unauthorized
* 403 Forbidden
* 404 Not Found
* 500 Internal Server Error
